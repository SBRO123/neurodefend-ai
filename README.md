# NeuroDefend AI 🛡️

**Detect. Protect. Empower.**

AI-Powered Phishing Detection & Real-Time Threat Monitoring for South Africa.

NeuroDefend AI detects phishing scams, fake bursary offers, banking fraud, fake job offers, WhatsApp scams, and suspicious account behavior in real-time using AI-powered analysis.

---

## 🚀 Quick Start

```bash
npm install
npm run dev
# Open http://localhost:3000
```
